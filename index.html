All Commands

?help - See all bot comands
?ban - Ban a member
?kick - Kick a member
?mute - Mute someone(muted role needed)
?unmute - Unmute someone
?addemoji - Add the given emoji
?slowmode - Set a channel slowmode
?lock - Lock a channel
?unlock - Unlock a channel
?warn - Warn a user
?check-warn - Check a user's warnings
?clearwarns - Clear 1 warning from a user
?nukewarns - Clear all warns of a user
?prefix - Set a server prefix
?rr - Make a reaction roles feature for the server
?chatbot - Set the chatbot channel for the server
?send-message - Send a message to the channel(you can use all placeholders)
?bot-logs - Set a server's bot logs
?nuke - Nuke a channel
?purge - Purge messages
?add-pw - Add a prohibited server word(if the word is sent, user gets warned and message is deleted. Only for users with no manage server permissions)
?pw-list - Show the servers prohibited words
?delete-pw - Delete a prohibited word

embed.author('Author text':'optional author image link') - Set an embed author

embed.title('Title text':'optional title link') - Set an embed title

embed.description('Text') - Set an embed description

embed.field('Field Header':'Field Value':'optional inline, yes or no') - Add an embed field

embed.footer('Text':'Optional image link') - Add an embed footer

embed.color('Name/Hex/RANDOM') - Add an embed color

embed.image('link') - Add and embed image

embed.thumbnail - Add an embed thumbnail

timestamp - Add an embed timestamp message.attachment('link') - Add a non embedded image

message.author - Get the author's ID

mentioned.id('number':'optional return author if no one is mentioned') - Get the mentioned user's id

random.user - Get a random user's ID from the server

role.give('User ID':'Role id') - Give the user a role

role.remove('user id':'role id') - remove a role from a user

if('condition') - Start an if condition. All if statements need to be closed with else()

end() - end an if condition

else() - start an else statement

replace('text':'what to replace':'replace with what':'optional how many times') - Replace what to replace with replace with what, from text

message.channel.send('channel id':'message') - send message to a channel

var.set('var':'value':'optional user') - Set a user's var for that server(vars can be ur[1-10]. Eg - ur1)

var.get('var':'optional user') - Get a user var value

var.server.set('var name':'value') - Set a var value for the server

var.server.get('var name') - Get a server var value

command.delete - Delete the user's command

reply.delete('time') - Delete the bot's reply in the given time

truncate('equation') - Round off an equation

text.split('separator') - Split a text

split.text('number') - Get a splitted text

user.name/nick/tag/discrim/avatar('user id') - Get a user's discriminator, avatar, tag, nickname or name

random.number('lower value':'higher value') - Get a random number from lower value to higher value

reactions.add('emoji 1':'2 etc') - Add reactions to the user's message

reactions.bot.add('emoji 1':'2 etc') - Add reactions to the bot's reply

server.name/icon - Get the server's name/icon

emoji('name') - Use an emoji that is not discord's basic emoji

members.count - Returns the guild's member count

perm/role.has('user id':'perm name/role id') - Checks if the user has the given role/perm

message.contains('message':'text':'more text etc') - Checks if the message contains the text

user.message - Get the user's message
